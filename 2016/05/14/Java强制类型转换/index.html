Java强制类型转换

  • Java,基础

在Java开发过程中，经常需要对对象进行转换，下面是几个将对象转换字符串;

1.对象转换字符串

  • toString

    String userStr=obj.toString(); 在使用之前一定要判断obj是否为空，否则一定会报NullPointerException异常;

  • （String）object

    String userStr=(String)obj; 在使用之前一定要判断instanceof类型检查是否能否转换，否则会报CalssCastException异常

  • String.valueOf(Object)

    String userStr=String.valueOf(obj);它的基础是Object.toString();,不过它已经对象为空进行判断

2.原生类型包装类转换

String idStr="12";
Integer idInt = Integer.valueOf(idStr);  //返回对象
int intValue = idInt.intValue(); //获取值
int parseInt = Integer.parseInt(idStr); //返回值    

其余的包装类都有对应的接口，都差不，在这里我就不一一列举了
